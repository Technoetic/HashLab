# Step 002 Context 전략 - Chunk 1: Step-Back 분석

## 조사의 핵심 목적

"해시 함수" 알고리즘 자료구조 튜토리얼을 **인터랙티브 웹**으로, **초보자 학습용**으로, **대중 앱 사례를 참고**해 **직관적으로** 이해할 수 있게 생성한다.

## 영향 범위

- Step 003~040: 디자인/사례/콘텐츠 조사 단계
- Step 041~080: 인터랙티브 시각화 구현 단계
- Step 081~114: 검증/배포 단계

## 핵심 항목 3가지

1. **해시 함수 핵심 개념** — 입력→고정 길이 출력, 결정성, 충돌, Avalanche 효과, One-Way
2. **대중 앱 사례** — 비밀번호 저장, Git 커밋, BitTorrent, 블록체인, IPFS, 카카오톡 메시지 무결성, Bloom Filter, HashMap
3. **인터랙티브 시각화 패턴** — 입력 슬롯, 실시간 해시 변환 애니메이션, 충돌 시뮬레이터, 해시 테이블 스토리지 비주얼라이저

## 프로젝트 규모

- 파일 수: 82개 (node_modules/.git/step_archive 제외)
- Scale: SMALL (<100 files)
- 권장 서브에이전트: 5~10개
