# Step 025 기획 chunk1 - 프로젝트 정의

## 프로젝트명
**HashLab** — 해시 함수 인터랙티브 학습 튜토리얼

## 한 줄 정의
초보자가 해시 함수의 본질·대중 앱 사례·해시 테이블 동작을 직접 손으로 만지며 30분 안에 직관적으로 이해하는 웹 튜토리얼.

## 대상 사용자
- 컴퓨터공학/SW 입문 학생
- "비밀번호 어떻게 저장하지?" 같은 개발자 호기심층
- 코딩 부트캠프 수강생

## 학습 목표(LO)
1. 해시 함수의 4대 성질을 자기 언어로 설명할 수 있다
2. 일상 앱에서 해시가 쓰이는 위치 5개 이상 든다
3. 해시 테이블 충돌과 해결책(체이닝/오픈 어드레싱)을 시각으로 인지한다
4. Avalanche 효과를 직접 체험으로 깨닫는다
5. 비밀번호는 평문 저장이 위험하고 해시(+salt)가 표준임을 안다

## 챕터 구조 (8장)
| # | 제목 | 인터랙션 | 시간 |
|:---|:---|:---|:---|
| 0 | 환영합니다 | 즉석 SHA-256 데모 | 1분 |
| 1 | 해시란 무엇인가 | 입력 슬롯 + 실시간 다이제스트 | 4분 |
| 2 | 4대 성질 | 결정성/Avalanche/단방향 체험 | 5분 |
| 3 | 비밀번호 저장 | 평문 vs 해시 vs +salt 비교 | 4분 |
| 4 | 해시 테이블 | 16버킷 시각화, 단어 입력 | 5분 |
| 5 | 충돌 처리 | 체이닝 vs 오픈 어드레싱 토글 | 4분 |
| 6 | Git 커밋 | 1바이트 변경 → 해시 변화 | 3분 |
| 7 | 비트코인 마이닝 | nonce 자동 증가 → 0 발견 | 3분 |
| 8 | 마무리 퀴즈 | 5문항 확인 학습 | 2분 |

## 핵심 인터랙티브 컴포넌트
- HashPlayground (입력 → SHA-256/SHA-1/MD5 즉시 출력)
- AvalancheCompare (1비트 차이 비교)
- HashTableViz (16버킷 + 카드 애니메이션)
- BloomFilterViz (32비트 점등)
- MiningSim (난이도 슬라이더 + nonce 카운터)
